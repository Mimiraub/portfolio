Library Room Booking System

This system enables UTeM students to book study rooms in the library. Features include:

  • Choose room types such as individual, group, and seminar rooms.
  • Select preferred date and time slots based on room availability.
  • Real-time room status updates to inform users of changes.
  • Cancel bookings by entering Matric ID.

Technologies & Concepts Used:

  • Developed in C++ using Visual Studio.
  • Applied Array-Based Lists to store room data.
  • Utilized Linked Lists to manage user bookings.
  • Implemented Queues to handle waiting lists.
  • Used Binary Search Trees (BST) for quick search via Matric ID.
  • Applied Hashing to represent paths between library areas.
