Movie Ticket System

This Movie Ticket System allows users to:

  • Pick movies based on genre and age rating.
  • Enter the number of persons based on different age such as citizen, kids and adult.
  • View seat map and do reservations based on availability.
  • Add food and beverages in order list.
  • Apply discount vouchers by entering the voucher code.

Built with C++ in Visual Studio.
